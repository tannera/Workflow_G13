Apache Camel Routing Project
=========================

Developed for 188.924 Workflow Modeling and Process Management, Vienna University of Technology.  

Using:
- Eclipse
- Maven
- Apache Camel
- Java 
- PostgreSQL (pgAdmin III)

With Eclipse Luna:
1. File - Import - General - Import existing projects into workspace.
2. After successful import, right-click on project ("Workflow_G13") -> Maven -> Update Project

With pgAdmin:
Create a new Database named Procurement. 

To run:
- Run as Java application -> MainApp
